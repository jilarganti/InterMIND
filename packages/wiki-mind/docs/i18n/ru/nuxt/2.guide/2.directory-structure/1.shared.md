Директория `shared/` позволяет вам делиться кодом, который может использоваться как в Vue приложении, так и на Nitro сервере.

::note
Директория `shared/` доступна в Nuxt v3.14+.
::

::important
Код в директории `shared/` не может импортировать любой Vue или Nitro код.
::

## Использование

**Метод 1:** Именованный экспорт

```ts twoslash [shared/utils/capitalize.ts]
export const capitalize = (input: string) => {
  return input[0] ? input[0].toUpperCase() + input.slice(1) : ''
}
```

**Метод 2:** Экспорт по умолчанию

```ts twoslash [shared/utils/capitalize.ts]
export default function (input: string) {
  return input[0] ? input[0].toUpperCase() + input.slice(1) : ''
}
```

Теперь вы можете использовать [автоматически импортируемые](/docs/guide/directory-structure/shared#auto-imports) утилиты в вашем Nuxt приложении и директории `server/`.

```vue [app.vue]
<script setup lang="ts">
const hello = capitalize('hello')
</script>

<template>
  <div>
    {{ hello }}
  </div>
</template>
```

```ts [server/api/hello.get.ts]
export default defineEventHandler((event) => {
  return {
    hello: capitalize('hello')
  }
})
```

## Как сканируются файлы

Только файлы в директориях `shared/utils/` и `shared/types/` будут автоматически импортированы. Файлы, вложенные в поддиректории этих директорий, не будут автоматически импортированы, если вы не добавите эти директории в `imports.dirs` и `nitro.imports.dirs`.

::tip
Способ работы и сканирования автоимпортов `shared/utils` и `shared/types` идентичен директориям [`composables/`](/docs/guide/directory-structure/composables) и [`utils/`](/docs/guide/directory-structure/utils).
::

:read-more{to="/docs/guide/directory-structure/composables#how-files-are-scanned"}

```bash [Структура директорий]
-| shared/
---| capitalize.ts        # Не автоимпортируется
---| formatters
-----| lower.ts           # Не автоимпортируется
---| utils/
-----| lower.ts           # Автоимпортируется
-----| formatters
-------| upper.ts         # Не автоимпортируется
---| types/
-----| bar.d.ts           # Автоимпортируется
```

Любые другие файлы, которые вы создаете в папке `shared/`, должны быть импортированы вручную с использованием алиаса `#shared` (автоматически настраивается Nuxt):

```ts
// Для файлов непосредственно в директории shared
import capitalize from '#shared/capitalize'

// Для файлов во вложенных директориях
import lower from '#shared/formatters/lower'

// Для файлов, вложенных в папку внутри utils
import upper from '#shared/utils/formatters/upper'
```

Этот алиас обеспечивает согласованные импорты во всем вашем приложении, независимо от расположения импортирующего файла.
