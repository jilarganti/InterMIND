# InterMIND 시작하기

InterMIND에 오신 것을 환영합니다! 이 게시물에서는 AI 기반 플랫폼의 기본 사항을 안내해 드립니다.

<!--more-->

## InterMIND란 무엇인가요?

InterMIND는 인공지능과 인간의 전문성을 결합하여 강력한 비즈니스 솔루션을 창출하는 혁신적인 플랫폼입니다.

## 주요 기능

- **AI 채팅 통합**: GPT-4와 Claude 기반
- **벡터 검색**: Upstash를 통한 시맨틱 검색 기능
- **다국어 지원**: 19개 이상의 언어 지원
- **OAuth 통합**: 안전한 인증 시스템
