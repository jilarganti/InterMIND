## Metadata

### Title

グローバルチームのためのリアルタイム音声翻訳 — **遅延なし**、**商談の損失なし**、**言語の壁なし**。

### Description

ビデオ通話のためのリアルタイム音声通訳。自分の言語で話し、全員の発言を即座に自然に理解 — 字幕も遅延もなく。
